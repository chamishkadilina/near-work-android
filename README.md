## ✨ Why NearWork?

Finding a job is hard. **NearWork makes it simple.**

- **See jobs near you** - on a map, not a list.
- **Know your chances** - your CV is scored against every job.
- **Apply in one tap** - then chat directly with recruiters.
- **Track every application** - receive live status updates.

> **Open the map. Find a job. Apply. Done.**

---

## 🚀 Get Started

Getting started takes less than a minute.

1. Download the APK using the button at the top.
2. Allow **Install from unknown sources** if Android asks.
3. Sign in with Google.
4. Start exploring jobs near you.
